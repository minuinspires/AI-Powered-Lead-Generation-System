# AI-Powered Lead Generation System

An intelligent Salesforce-based application that automates lead capture, qualification, scoring, and prioritization using Salesforce Agentforce, Prompt Builder, and Flow Automation.

## Project Overview

The AI-Powered Lead Generation System helps organizations identify high-quality sales leads by analyzing customer information, website interactions, and business data. Using Salesforce AI capabilities, the system predicts the likelihood of lead conversion, generates AI-powered summaries, and automatically notifies sales representatives for timely follow-up.

## Workflow

1. Capture leads from multiple sources.
2. Store lead details in Salesforce.
3. Validate lead information.
4. Calculate AI-based lead score.
5. Generate AI summary using Prompt Builder.
6. Trigger automated workflows.
7. Notify sales representatives.
8. Track lead progress.
9. Generate reports and dashboards.
