**Developed By:** Minu Antony

# Project Overview

The **AI-Powered Lead Generation System** is a Salesforce-based intelligent CRM solution designed to automate lead capture, qualification, prioritization, and follow-up using Salesforce Agentforce, Prompt Builder, and Flow Automation. The system helps sales teams identify high-potential leads, improve response times, and increase conversion rates through AI-driven decision-making.

---

# Project Objectives

- Automate lead capture from multiple sources.
- Prioritize leads using AI-powered lead scoring.
- Generate AI-based lead summaries.
- Automate sales workflows and notifications.
- Improve sales efficiency and customer engagement.
- Reduce manual effort in lead management.
- Provide actionable insights through reports and dashboards.

---

# Project Scope

The project focuses on developing an intelligent lead management solution within Salesforce. The system includes:

- Lead Management
